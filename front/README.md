# TAP ARENA

## Версии 
`node => 20.14.0`
`npm => 10.7.0`

## Порядок запуска `dev` версии

### Инициализация пакетов
```sh
npm install
```

### Открытие приложение на локальном сервере
```sh
npm run dev
```

### Создание автообновляемого билда при `dev` значениях
```sh
npm run serve-dev
```

## Порядок запуска `prod` версии

### Инициализация пакетов
```sh
npm install
```

### Создание билда без автообновления
```sh
npm run build-only
```
